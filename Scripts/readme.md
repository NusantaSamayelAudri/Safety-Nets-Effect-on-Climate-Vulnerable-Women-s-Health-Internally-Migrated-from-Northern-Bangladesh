# Scripts

This folder contains the R analysis workflow for the study.

## Main Script

[`01_clean_analyze_model.R`](01_clean_analyze_model.R)

This script documents the full quantitative analysis workflow:

- Data import
- Data cleaning
- Variable construction
- Descriptive summaries
- Hypothesis-based bivariate tests
- Logistic regression modeling
- Odds-ratio table generation
- Figure exports

The raw survey dataset is not publicly shared due to respondent confidentiality, but the script is included to document the reproducible analysis logic used in the study.
