## Data Source

This repository is based on primary mixed-methods fieldwork conducted for the study **“Safety Nets Effect on Climate-Vulnerable Women’s Health: Internally Migrated from Northern Bangladesh.”**

The quantitative analysis uses household survey data collected from **120 climate-displaced women-headed or women-represented households** across three informal settlements in Dhaka, Bangladesh:

| Study Site | Sample Size |
|---|---:|
| Korail Slum | 40 households |
| Kallyanpur Porah Slum | 40 households |
| City Colony Slum | 40 households |
| **Total** | **120 households** |

The sample was selected through a stratified household survey design, with proportional allocation across sub-areas within each slum. The broader study also included qualitative fieldwork, including in-depth interviews, focus group discussions, and key informant interviews, to contextualize the quantitative findings.

### Data Coverage

The dataset includes variables related to:

- climate-related displacement pathways
- migration history and duration of residence in Dhaka
- social safety-net and aid receipt
- health-related aid and clinic access
- self-reported health conditions
- income before and after migration
- employment before and after migration
- education and hygiene indicators
- household-level socioeconomic vulnerability

### Data Privacy

The raw household-level dataset is **not publicly shared** because it contains sensitive respondent-level information from climate-displaced women living in informal settlements. This repository therefore provides the analysis workflow, selected figures, variable-construction logic, and model documentation without exposing confidential respondent records.
